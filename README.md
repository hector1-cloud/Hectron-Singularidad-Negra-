## ⚡ MANIFIESTO DEL SISTEMA
**HECTRON UI v1.0** es el nodo de control diseñado para la gestión de proyectos de IA de **AbadaLabs**. Esta interfaz materializa la filosofía de **Fricción Cero**, permitiendo a los operadores configurar arquitecturas de vanguardia (Chatbots, Predictores y Clasificadores) bajo el amparo del núcleo HECTRON.

## 🛠️ ARQUITECTURA DE LA MATRIZ
El sistema está fragmentado en componentes reactivos para asegurar una carga modular:

* **`App.jsx`**: El Sistema Nervioso Central (Gestor de Navegación).
* **`PageInicio.jsx`**: Protocolo de entrada y bienvenida.
* **`PageSeleccionProyecto.jsx`**: Bifurcación de lógica de negocio.
* **`PageConfiguracionIA.jsx`**: Inyección de modelos de Hugging Face.
